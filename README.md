# LayerLook 👗

LayerLook is an AI-powered fashion storytelling tool that helps creators transform outfit photos, accessories, fit checks, and fashion references into polished editorial-style collages optimized for social media. Built during the UPSCALE Conference Hackathon.

Built by **Team Laputa** (Peggy Shen & Bernice Sun) for the **UPSCALE Conference Hackathon — June 2026**.

## The Problem

Fashion creators spend a surprising amount of time creating visual stories.

A typical Instagram Story outfit collage might require:
* collecting outfit photos
* gathering Pinterest references
* removing backgrounds
* cutting out accessories
* arranging layouts
* resizing for social platforms
* experimenting with aesthetics

While collecting inspiration is easy, turning that inspiration into a polished post is still largely manual.

LayerLook helps bridge that gap.

---

## Our Solution

LayerLook transforms fashion assets into social-ready visual stories.

Users can:

1. Upload outfit photos and fashion references
2. Choose a creative direction
3. Generate a polished fashion composition

The same outfit can be interpreted through multiple visual styles, helping creators quickly explore different aesthetics.

---

## How it works

### Fashion Story Generation

Transform outfit photos into curated visual compositions.

### Background Removal

Automatically isolate fashion subjects using Google Gemini-powered image processing.

### Multiple Creative Directions

Generate stylistically distinct outputs:
* Editorial magazine layouts
* Scrapbook collages
* Visual journals
* Modernist compositions
* Social-first fashion stories

### Social Media Optimization

Designed for content formats such as:
* Instagram Stories
* Outfit recaps
* Fashion moodboards
* Pinterest-style collages
